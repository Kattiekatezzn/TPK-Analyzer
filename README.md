# TPK Analyzer (Style B)

Minimal flat card casino theme version.

## How to run locally
1. Put these files (`index.html`, `style.css`, `script.js`, `service-worker.js`, `manifest.json`) in a folder.
2. Open `index.html` in your browser.

## GitHub Pages
- Upload files to a public repository root.
- In repo Settings → Pages → set source to `main` branch, `/ (root)`.
- After a moment your site is available at `https://<username>.github.io/<repo>/`.
